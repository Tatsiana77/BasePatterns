Шаблон: Прокси (Proxy)
==============================================
Цель:
Для замещения другого обьекта и контроля доступа к нему
===============================================

Используется для обеспечения контроля доступа к определенному обьекту.

===============================================
Пример использования :

- когда есть необходимость ссылаться на обьект способом отличным от обычного указателя;
* удаленное замещение - локальный представитель вместо обьекта в другом
                        адресном пространстве.
* виртуальное замещение - создание "тяжелых объектов при необходимости".
* защищеное замещение - контроль доступа к указонному объекту.
* "умная ссылка"  - замена обычного указателя (подсчет числа ссылок,установка блокировки и тд.)
